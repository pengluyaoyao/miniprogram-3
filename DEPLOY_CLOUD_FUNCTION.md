# 部署云函数指南

## 方法1：通过开发者工具部署（推荐）

1. 打开微信开发者工具
2. 在左侧文件树中找到 `cloudfunctions/aiAnalysis` 文件夹
3. **右键点击** `aiAnalysis` 文件夹
4. 选择「上传并部署：云端安装依赖」
5. 等待部署完成（会显示"上传成功"）

## 方法2：通过云开发控制台部署

1. 打开微信开发者工具
2. 点击顶部的「云开发」按钮
3. 在左侧选择「云函数」
4. 找到 `aiAnalysis` 函数
5. 点击右侧的「更多」→「重新部署」

## 部署成功的标志

在开发者工具的控制台会显示：
```
[云函数] [aiAnalysis] 上传成功
[云函数] [aiAnalysis] 部署成功
```

## 测试步骤

部署完成后，按以下步骤测试：

### 1. 清除旧数据（重要！）
- 删除之前测试的任务记录
- 或使用新账号测试

### 2. 完整测试流程
1. 打开小程序（开发版）
2. 上传户型图
3. 点击「开始分析」
4. **看到授权弹窗，点击【允许】**
5. 等待分析完成（10-30秒）
6. 打开云开发控制台→云函数→aiAnalysis→日志

### 3. 查看关键日志

应该看到以下日志（带emoji）：

### 4. 检查微信消息

- 打开微信
- 进入「服务通知」
- 应该能看到「户型风水分析」的消息
- 点击消息，应该跳转回小程序并显示分析结果

## 常见部署问题

### Q1: 部署失败，提示权限错误
**解决**：确保在云开发控制台中该云函数已创建

### Q2: 部署成功但功能不生效
**解决**：清除小程序缓存，重新编译

### Q3: 日志看不到新的输出
**解决**：确认是否真的重新部署了，检查部署时间
