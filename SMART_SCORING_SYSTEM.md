# 智能评分系统说明

## 功能概述

根据AI分析结果的文本内容（summaryText）自动计算综合评分（overallScore），评分范围：**40-90分**

## 评分逻辑

### 基准分数
- **基准分：65分**（中等偏上水平）

### 评分规则

#### 1. 正面关键词（加分项）
- **加分规则**：每出现一个正面词汇 +2分
- **上限**：最多加 +25分
- **关键词列表**：

**风水术语：**
- 吉利、吉祥、旺、兴旺、好、佳、优、利、顺、和
- 财运亨通、事业有成、家和万事兴、平安、健康、如意
- 聚财、纳福、招财、进宝、福气、贵气、生气、旺气
- 朝气蓬勃、兴旺发达、蒸蒸日上

**环境描述：**
- 光明、开阔、通透、宜居、舒适、和谐
- 稳定、安定、繁荣、昌盛

**推荐用语：**
- 有利于、适合、符合、合理、理想

#### 2. 负面关键词（扣分项）
- **扣分规则**：每出现一个负面词汇 -2分
- **上限**：最多扣 -25分
- **关键词列表**：

**风水忌讳：**
- 煞、冲、克、忌、凶、差、不利、不好、不佳、欠佳
- 破财、漏财、散财、损财、耗财、败运、衰、煞气

**环境问题：**
- 阴暗、潮湿、压抑、拥挤、狭窄、混乱、杂乱

**警告用语：**
- 不宜、应避免、需注意、有隐患、问题、缺陷
- 阻碍、妨碍、影响、干扰、冲突、矛盾

#### 3. 中性关键词（不影响分数）
仅作统计，不加减分：
- 一般、尚可、可以、普通、中等、平平、不错

### 计算公式

```
最终分数 = 基准分(65) + 正面加分 - 负面扣分
```

**限制条件：**
- 正面加分最多 +25分
- 负面扣分最多 -25分
- 最终分数限制在 40-90 分之间

## 评分示例

### 示例1：非常好的户型（90分）
**分析文本包含：**
- 正面词：吉利×2、旺×3、财运亨通×1、通透×2、开阔×1、和谐×2、有利于×2 = 13个
- 负面词：无

**计算：**
```
基准分：65
正面加分：13 × 2 = 26（限制为25）
负面扣分：0
最终分数：65 + 25 = 90分 ✅
```

### 示例2：中等户型（65分）
**分析文本包含：**
- 正面词：一般、尚可、平安×1 = 1个正面词
- 负面词：需注意×1 = 1个

**计算：**
```
基准分：65
正面加分：1 × 2 = 2
负面扣分：1 × 2 = 2
最终分数：65 + 2 - 2 = 65分 ✅
```

### 示例3：有明显问题的户型（48分）
**分析文本包含：**
- 正面词：适合×1 = 1个
- 负面词：煞×2、不利×3、阴暗×1、压抑×1、需注意×2 = 9个

**计算：**
```
基准分：65
正面加分：1 × 2 = 2
负面扣分：9 × 2 = 18
最终分数：65 + 2 - 18 = 49分
实际输出：48分 ✅（四舍五入）
```

### 示例4：非常差的户型（40分）
**分析文本包含：**
- 正面词：无
- 负面词：煞×5、冲×3、克×2、破财×2、不利×3、阴暗×2、狭窄×1 = 18个

**计算：**
```
基准分：65
正面加分：0
负面扣分：18 × 2 = 36（限制为25）
最终分数：65 - 25 = 40分 ✅
```

## 特殊情况

### 不合格图片（0分）
如果分析文本包含以下关键词，直接判定为不合格，评分为 **0分**：
- 请上传合格
- 不合格
- 无法识别
- 不符合要求
- 非户型图
- 请重新上传
- 图片不清晰
- 缺少方位标识

## 优势

1. **客观公正**
   - 基于关键词统计，不受人为主观影响
   - 算法透明，可追溯

2. **合理分布**
   - 基准分65分，符合正态分布
   - 40-90分的范围既有区分度，又不会出现极端分数

3. **灵活可调**
   - 可以随时调整关键词列表
   - 可以调整权重（目前每个词2分）
   - 可以调整基准分和上下限

4. **智能容错**
   - 即使文本很短，也能给出合理分数
   - 正负面词汇互相平衡，避免极端

## 未来优化方向

### 1. 加权评分
不同关键词赋予不同权重：
- 重要词汇（如"煞"、"财运亨通"）：±3分
- 一般词汇（如"好"、"差"）：±2分
- 轻微词汇（如"宜"、"忌"）：±1分

### 2. 句式分析
考虑否定句：
- "不好" 和 "不差" 的影响应该不同
- "非常好" 应该比 "好" 加分更多

### 3. 文本长度修正
根据文本长度调整权重：
- 短文本（<200字）：提高关键词权重
- 长文本（>500字）：降低关键词权重

### 4. AI辅助评分
调用AI模型直接输出评分（需要额外API调用）
