# 分享标题生成策略

## 设计理念

基于小红书爆款内容的特点，生成**吸引眼球、避免迷信、突出实用性**的分享标题。

## 核心原则

### ✅ 应该做的

1. **制造悬念和好奇心**
   - 使用"必须注意"、"后悔没早看"等表述
   - 突出实测、真实体验

2. **突出实用价值**
   - 强调避坑、优化、实用性
   - 提供具体的户型信息

3. **使用情感共鸣**
   - "让人羡慕"、"太惊艳"、"绝了"
   - 真实感受的表达

4. **数字和对比**
   - 户型面积、房间数
   - 前后对比、优缺点对比

### ❌ 应该避免的

1. **迷信词汇**
   - ❌ 风水、煞、冲、克
   - ❌ AI、智能（过于科技化）
   - ❌ 吉利、旺、财运等传统词汇

2. **夸大宣传**
   - ❌ "完美"、"最好"
   - ❌ 绝对化表述

3. **生硬广告**
   - ❌ 直接推销
   - ❌ 商业化太明显

## 标题模板分类

### 1. 测评体验类（最常用）

**特点**：强调真实体验，建立信任感

```
✨ 实测！3室2厅户型住了3个月的真实感受
🏠 3室2厅户型测评｜这些细节必须注意！
💯 3室2厅户型深度解析｜避坑必看
🎯 3室2厅户型实测｜优缺点全分析
```

**适用场景**：所有户型

### 2. 优化指南类

**特点**：提供价值，解决问题

```
🔥 3室2厅户型优化指南｜后悔没早看
📐 3室2厅98㎡｜户型优化全攻略
💡 3室2厅户型布局｜实用性拉满
🌟 3室2厅户型改造｜前后对比太惊艳
```

**适用场景**：有明显优化空间的户型

### 3. 避坑提醒类

**特点**：引发焦虑，提供解决方案

```
⚠️ 3室2厅户型踩坑记｜这些问题要注意
🚨 3室2厅户型｜这个缺陷千万别忽视
💥 3室2厅户型｜住进来才发现的问题
```

**适用场景**：有明显缺点的户型

### 4. 惊喜发现类

**特点**：制造反差，激发好奇

```
👀 3室2厅户型｜通透+舒适绝了
✨ 3室2厅户型｜居然这么通透！
💫 3室2厅户型｜通透到让人羡慕
🎊 3室2厅体验｜通透的宝藏户型
```

**适用场景**：有明显亮点的户型

### 5. 面积专属类

**特点**：针对性强，精准定位

**小户型（<70㎡）**：
```
🏠 65㎡小户型｜空间利用率超高
✨ 65㎡3室1厅｜小而精致的秘密
💡 60㎡户型｜这样布局才不挤
```

**刚需户型（90-120㎡）**：
```
🏠 98㎡3室2厅｜刚需户型深度测评
✨ 98㎡户型｜舒适度满分
💯 100㎡户型｜性价比超高
```

**大户型（>120㎡）**：
```
🏠 140㎡4室2厅｜大户型优化方案
✨ 150㎡户型｜空间布局太合理
🌟 130㎡户型｜豪宅级体验
```

## 关键词替换策略

### 避免迷信词汇的替换方案

| ❌ 避免使用 | ✅ 替换为 |
|------------|---------|
| 风水 | 户型优化、布局分析 |
| AI | 专业、智能（谨慎使用）|
| 吉利、旺 | 通透、舒适、明亮 |
| 财运 | 实用性、性价比 |
| 煞、冲 | 问题、缺陷、需注意 |
| 福气 | 舒适感、幸福感 |

## 标题生成算法

### 流程图

```
输入：户型信息（房间数、面积）、分析结果
    ↓
提取关键词：通透、舒适、明亮等
    ↓
判断面积范围：小户型/刚需/大户型
    ↓
生成标题池：基础模板 + 亮点模板 + 面积模板
    ↓
随机选择一个标题
    ↓
输出：吸引人的小红书风格标题
```

### 代码实现

```typescript
generateShareCopy() {
  // 1. 提取信息
  const rooms = formData.rooms || '温馨户型';
  const area = formData.area || '';
  const summary = analysisResult.summary;
  
  // 2. 提取关键词（避免迷信词汇）
  const positiveKeywords = [
    '通透', '开阔', '舒适', '和谐', '明亮', 
    '宽敞', '采光', '布局', '合理', '实用'
  ];
  const highlights = positiveKeywords
    .filter(k => summary.includes(k))
    .slice(0, 3);
  
  // 3. 基础标题模板（10个）
  const titleTemplates = [
    `🏠 ${rooms}户型测评｜这些细节必须注意！`,
    `✨ 实测！${rooms}户型住了3个月的真实感受`,
    // ... 更多模板
  ];
  
  // 4. 亮点标题（5个，条件生成）
  if (highlights.length > 0) {
    titleTemplates.push(
      `✨ ${rooms}户型｜${highlights[0]}到让人羡慕`
    );
  }
  
  // 5. 面积标题（6个，条件生成）
  if (area) {
    const areaNum = parseInt(area);
    if (areaNum < 70) {
      titleTemplates.push(
        `🏠 ${area}㎡小户型｜空间利用率超高`
      );
    }
  }
  
  // 6. 随机选择
  return titleTemplates[random];
}
```

## 注意事项

### 1. 避免标题党

❌ **不要**：
- 夸大其词（"史上最强"）
- 误导性表述（"100%完美"）
- 制造焦虑（"不看后悔一辈子"）

✅ **应该**：
- 真实客观（"实测体验"）
- 有价值（"避坑指南"）
- 适度吸引（"值得一看"）

### 2. 词汇合规性

定期检查标题模板，确保：
- 不含迷信词汇
- 符合平台规范
- 不涉及敏感词

### 3. 用户反馈

收集用户反馈，优化标题模板：
- 哪些标题点击率高
- 哪些词汇更受欢迎
- 哪些风格更符合目标用户

## 未来优化方向

### 1. 智能化

根据用户画像和历史数据：
- 年龄段偏好不同风格
- 区域差异调整用词
- 时间段优化标题

### 2. 个性化

根据具体户型特点：
- 朝向好 → 突出采光
- 面积大 → 强调空间感
- 新装修 → 突出设计感

### 3. 季节性

根据季节调整关键词：
- 夏季：通风、凉爽
- 冬季：保暖、舒适
- 春秋：舒适、宜居
