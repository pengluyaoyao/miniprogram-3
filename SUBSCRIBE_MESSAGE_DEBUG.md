# 订阅消息测试与排查指南

## 为什么收不到订阅消息？

### 排查步骤

#### 1. ✅ 检查用户是否授权

**测试方法**：
在小程序中点击"开始分析"时，是否弹出了订阅消息授权弹窗？

```
┌──────────────────────────┐
│   订阅消息               │
│                          │
│ "户型风水分析"           │
│  需要向你发送订阅消息    │
│                          │
│  [取消]      [允许]      │
└──────────────────────────┘
```

**注意**：
- 必须点击【允许】才能收到消息
- 如果点击【取消】，不会收到消息（但分析会正常进行）
- 每次授权只能发送一次消息

---

#### 2. ✅ 检查云函数日志

这是最重要的排查手段！

**查看步骤**：
1. 打开微信开发者工具
2. 点击「云开发」按钮
3. 选择「云函数」→ `aiAnalysis`
4. 点击「日志」标签

**应该看到的日志**：

✅ **成功的情况**：
```
用户openid: oXXXX-XXXXXXXX
准备发送订阅消息: { openid: 'oXXXX', taskId: 'xxx' }
订阅消息发送结果: { errCode: 0, errMsg: 'openapi.subscribeMessage.send:ok' }
```

❌ **失败的情况1 - 字段不匹配**：
```
发送订阅消息失败: Error
错误代码: 40003
错误信息: invalid openid
```

❌ **失败的情况2 - 用户未授权**：
```
发送订阅消息失败: Error
错误代码: 43101
错误信息: user refuse to accept the msg
```

❌ **失败的情况3 - 模板字段错误**：
```
发送订阅消息失败: Error
错误代码: 47003
错误信息: argument invalid! data.thing1.value invalid
```

---

#### 3. ✅ 检查模板字段是否匹配

**当前代码使用的字段**：
```javascript
data: {
  thing1: { value: '户型风水分析' },
  time2: { value: '2025-11-06 15:30' },
  thing3: { value: '您的户型分析已完成，点击查看' }
}
```

**必须确认**：
- 您的模板实际字段名是 `thing1`、`time2`、`thing3` 吗？
- 还是 `thing4`、`date5`、`thing6`？
- 字段顺序是否对应？

---

#### 4. ✅ 检查云函数权限

云函数需要有调用 `openapi` 的权限。

**检查方法**：
1. 打开云开发控制台
2. 选择「云函数」→ `aiAnalysis`
3. 点击「权限配置」
4. 确保开通了「服务端调用」权限

---

#### 5. ✅ 检查开发环境设置

当前代码配置：
```javascript
miniprogramState: 'developer'  // 开发版
```

**测试要求**：
- 必须使用开发版小程序
- 不能使用体验版或正式版（除非改为 `'trial'` 或 `'formal'`）

## 常见错误代码

| 错误代码 | 错误原因 | 解决方法 |
|---------|---------|---------|
| 40003 | openid 无效 | 检查 openid 是否正确获取 |
| 43101 | 用户拒绝接收消息 | 用户点击了"取消"，需要重新授权 |
| 47001 | 参数缺失或格式错误 | 检查模板字段是否完整 |
| 47003 | 模板字段值不符合规范 | 检查字段值长度、格式 |
| 41030 | page 路径不正确 | 检查页面路径是否在 app.json 中注册 |

---

## 字段值的限制

微信对订阅消息字段有严格限制：

| 字段类型 | 最大长度 | 说明 |
|---------|---------|------|
| thing   | 20 字符 | 普通文本 |
| time    | 不限    | 时间格式：2025-11-06 15:30:00 |
| date    | 不限    | 日期格式：2025-11-06 |
| number  | 32 字符 | 数字 |
| letter  | 32 字符 | 字母或数字 |
| symbol  | 5 字符  | 符号（如金额） |
| phrase  | 5 字符  | 短语 |

⚠️ **当前代码中的问题**：
```javascript
thing3: { value: '您的户型分析已完成，点击查看' }  // 18个字符，符合
```
