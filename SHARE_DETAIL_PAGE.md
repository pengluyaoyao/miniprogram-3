# 分享详情页功能说明

## 功能概述

创建了一个**分享详情页**，用户点击分享卡片后会先进入这个精美的详情页，查看完整的分析文案后，再点击按钮跳转到分析页面。

## 用户体验流程

```
用户A完成户型分析
    ↓
点击"分享给好友"按钮
    ↓
分享给好友B
    ↓
好友B点击分享卡片
    ↓
打开小程序 → 进入"分享详情页" ✨
    ↓
查看完整的分享内容：
  - 好友分享标识
  - 吸引人的标题
  - 户型概况（户型/面积/朝向）
  - 户型亮点（通透/舒适等）
  - 专业分析摘要
    ↓
页面底部看到引导卡片：
  "💡 想了解你家户型的优缺点？"
    ↓
点击"立即体验专业测评"按钮
    ↓
跳转到分析页面 → 开始自己的户型测评
```

### 参数传递

**分享时携带的参数**：

```typescript
{
  title: '标题',              // 分享标题
  rooms: '3室2厅',           // 户型
  area: '98',                // 面积
  orientation: '南北通透',    // 朝向
  highlights: '["通透","舒适","明亮"]',  // 亮点（JSON）
  summary: '分析摘要文本...'  // 分析摘要（150字）
}
```

**URL示例**：
```
/pages/share-detail/share-detail?
  title=%E5%AE%9E%E6%B5%8B...&
  rooms=3%E5%AE%A42%E5%8E%85&
  area=98&
  highlights=%5B%22%E9%80%9A%E9%80%8F%22%5D&
  summary=...
```

### 核心代码

#### 1. 分享配置（analysis.ts）

```typescript
onShareAppMessage() {
  const copy = this.generateShareCopy();
  
  // 构建URL参数
  const params: string[] = [];
  params.push(`title=${encodeURIComponent(copy.title)}`);
  params.push(`rooms=${encodeURIComponent(formData.rooms)}`);
  // ... 其他参数
  
  const sharePath = `/pages/share-detail/share-detail?${params.join('&')}`;
  
  return {
    title: copy.title,
    path: sharePath,  // 指向分享详情页
    imageUrl: formData.imageUrl
  };
}
```

#### 2. 参数接收（share-detail.ts）

```typescript
onLoad(options: any) {
  // 解析URL参数
  if (options.title) {
    this.setData({
      shareTitle: decodeURIComponent(options.title)
    });
  }
  
  // 解析亮点列表
  if (options.highlights) {
    const highlights = JSON.parse(
      decodeURIComponent(options.highlights)
    );
    this.setData({ highlights });
  }
  
  // ... 其他参数
}
```

#### 3. 跳转到分析页

```typescript
goToAnalysis() {
  wx.redirectTo({
    url: '/pages/analysis/analysis',
    fail: () => {
      // 如果是Tab页面，使用switchTab
      wx.switchTab({
        url: '/pages/analysis/analysis'
      });
    }
  });
}
```

## 注意事项

### 1. URL长度限制

微信小程序的分享路径有长度限制（约256字符）：
- ✅ 已做优化：摘要限制150字
- ✅ 只传递核心信息
- ⚠️ 如果参数过长，考虑使用短链或存储到数据库

### 2. 参数编码

必须使用 `encodeURIComponent` 和 `decodeURIComponent`：
```typescript
// 分享时
params.push(`title=${encodeURIComponent(title)}`);

// 接收时
const title = decodeURIComponent(options.title);
```

### 3. 分享链路完整性

确保三个页面配置正确：
1. ✅ `analysis.ts` - 分享配置
2. ✅ `share-detail.ts` - 详情页逻辑
3. ✅ `app.json` - 页面注册

### 4. 回退策略

详情页的跳转使用了回退策略：
```typescript
wx.redirectTo({
  url: '/pages/analysis/analysis',
  fail: () => {
    // Tab页面用switchTab
    wx.switchTab({
      url: '/pages/analysis/analysis'
    });
  }
});
```
